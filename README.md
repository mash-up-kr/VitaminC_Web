# [Korrk](https://www.korrk.kr/)

**꼬르륵(Korrk)** 은 **20,30 💼 직장인**들이 지인들과 함께 만드는 🍔 **음식점 지도**를 통해 **손쉬운 🙌 식당 선택**을 돕는 서비스입니다.

## User Flow

```mermaid
flowchart TD
    A[www.korrk.kr] --> B{기회원}
    B -->|X| C[회원가입]
    B -->|O| D[로그인]
    C --> E[맛집 지도]
    D --> E[맛집 지도]

    E --> F[초대] --> A
    E --> G[맛집 검색]

    H --> B
```
